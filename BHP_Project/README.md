# BHP (Basic Handling of Python) Project

This project explains the basics of Python data handling.
It covers fundamental operations using Pandas and NumPy,
along with simple visualizations using Matplotlib.

**Note:** The dataset used in this project is **self-created** to demonstrate
the Basic Handling of Python (BHP) methods.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook
