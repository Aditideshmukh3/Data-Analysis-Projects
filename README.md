# Data Analysis Projects

This repository contains six data analysis projects created using Python.
Each project demonstrates different levels of data handling, analysis,
feature engineering, and visualization using Pandas, NumPy, and Matplotlib.

## Projects Included
1. BHP (Basic Handling of Python)
2. Employee Data Analysis
3. Titanic Data Analysis
4. Customer Churn Analysis
5. Netflix Data Analysis
6. Global Superstore Sales Analysis
